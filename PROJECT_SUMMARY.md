## 🎨 Design System

### Color Palette
```
Primary:    #c68740  (Gold/Bronze - Brand color)
Dark:       #0b0b0f  (Charcoal)
Accent:     #d4a574  (Light Gold)
Light BG:   #fdfcfb  (Off-white)
Text:       #6b7280  (Muted gray)
```

### Typography
- **Headings**: Responsive sizing (clamp)
- **Body**: 1.6 line-height for readability
- **Accents**: Letter-spacing for elegance
- **Hierarchy**: Clear size differentiation

### Spacing System
```
Small:   1rem    (g-3, p-3)
Medium:  1.5rem  (g-4, p-4)
Large:   3rem    (g-5, p-5)
```

### Shadow System
```
Small:   0 4px 12px rgba(0,0,0,0.08)
Medium:  0 10px 25px rgba(0,0,0,0.1)
Large:   0 20px 50px rgba(0,0,0,0.12)
```
